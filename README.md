#  Forge Fitness – Strength Training Gym Website

A modern and responsive gym website developed as **DecodeLabs Frontend Development Project 3**. This project allows users to explore gym facilities, membership plans, trainers, and request a free trial class through an integrated booking system connected to a **Node.js + Express.js + PostgreSQL backend**.

---

##  Live Demo

**Frontend:**  
https://forge-fitness-iqra.netlify.app/

---

##  Project Overview

Forge Fitness is a responsive gym website built with HTML, CSS, and JavaScript. The website provides users with information about gym services, membership plans, trainers, and allows them to book a free trial class. Form submissions are processed by a Node.js backend and stored in a PostgreSQL database.
